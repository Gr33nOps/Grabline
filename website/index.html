GrabLine

One file. Many lines. Done sooner.

A free download manager that splits each transfer across parallel connections, resumes after a crash, and pairs with your browser when you want a quality pick.

Download for your system Firefox extension

Latest release · Windows · macOS · Linux · AGPL-3.0

The idea

Grab the line. Finish the file.

Browsers open one pipe. GrabLine opens many — up to 128 — and steals work from slow segments so idle connections stay busy. If the power cuts, progress is already on disk. The dashboard shows that speed live.

Queue

The downloads window.

Live speed, ETA, and status in one table. Filter active or completed, search by name, open a details drawer when you need the graph.

Video & audio

Pick the quality before it starts.

From the browser button or a pasted link: resolution, MP3/M4A/FLAC, subtitles, chapters, SponsorBlock, optional clip. Powered by yt-dlp for 1000+ sites.

Also in the box

Beyond the queue.

Same app for torrents, cloud boxes, archives and the boring network controls.

Browser button

Hover media, right-click links, hand off over Native Messaging — no open ports.

Torrents

Magnets and .torrent files via libtorrent: DHT, sequential streaming, seed ratios, RSS.

Cloud & servers

SFTP, FTP, S3, WebDAV with keychain logins; Drive and Dropbox share links as direct pulls.

Queues & schedules

Named queues, priorities, dependencies, quiet hours, shut down when the list is done.

Proxies & caps

HTTP/SOCKS for downloads and torrents. Global, per-job and per-host speed limits.

Advisory security

Checksums, optional local AV / VirusTotal. Warnings never block — the file stays yours.

Honest limits

What it will not do.

No DRM bypass

Netflix, Prime, Disney+, Spotify tracks and the rest are refused with a clear message.

No login tricks

Optional browser-session uses your own login for your content. Cookies are not uploaded to us.

No usage telemetry

No ads, no accounts. Downloads hit the sites you ask for; a quiet GitHub update check is the only maintenance call.

FAQ
Is it free?

Yes. AGPL-3.0, no paid tier, no trial. Source is on GitHub.

How is it different from IDM?

Same core idea — many connections and a browser button — but GrabLine is open source, runs on macOS and Linux too, and includes video/audio, torrents, cloud and archives in one app. Nothing is sent to us.

Does YouTube work?

Yes. Hover, click the GrabLine button, pick quality (up to 4K) or audio (MP3/M4A/FLAC), with subtitles and trim. A thousand-plus other sites via yt-dlp.

Why does my OS warn me?

Unsigned installers. The code is public and built in CI. Use the click-through in the install guide; it is once per version.
